About

  • This module provides a utility function to evaluate the strength of a given password and offers suggestions for improvement.
  • Anyone with the desire to include password checks for their registered users can use it.
  • Its simple, free and also occupies less memory.

Features

  • Checks if the password meets the following criteria:
    • Minimum length of 8 characters
    • Contains at least one uppercase letter
    • Contains at least one lowercase letter
    • Contains at least one digit
    • Contains at least one special character
  • Returns a strength level and boolean validity.
  • Provides suggestions for improvement when the password does not meet specific criteria.

API Reference

  • Function:
    checkStrength(password)
  • Parameters:
    • password (string):
      The password to evaluate.
    • case-sensitive (boolean):
      The value which describes if password's language is case sensitive or not
  • Returns:
    An object containing:
  • booleanValue (boolean):
    Whether the password meets all criteria.
    • true - Password meets all criteria
    • false - Password does not meet one or more criteria
  • strength (number):
    Password strength level:
    • 0 - Weak
    • 1 - Moderate
    • 2 - Strong
  • suggestions (array):
    Suggestions for improving the password.
  • Dependencies:
    password_suggestions:
    An external module that provides suggestions for password improvements. Ensure the module exports properties like:
    • lengthCheck
    • capitalLetterCheck
    • smallLetterCheck
    • digitCheck
    • specialCharacterCheck

Installation Code

Install the package using the below code:

Usage

  • Import the checkStrength function into your project:

    const checkStrength = require('./checkStrength');

  • Call the function with a password string:

    const result = checkStrength('MySecureP@ss24'); console.log(result);

  • Output Example:

    • json
    • {
      • "booleanValue": true,
      • "strength": 2,
      • "suggestions": []
    • }

Code Samples

Following are the code samples for different type of passwords:
  • Password strength: 0
    • This type of strength is displayed when none of the characteristics are met or only 1 or 2 characteristics are met.
    • Note: If language is case sensitive then both capital and small letters check is included. Else only small letter check is included among the two
    • Example: abcd
    • Code sample:
      • const result = checkStrength('abcd');
      • console.log(result);
      • Output:
        • Strength: 0
        • Password should contain minimum of 8, and maximum of 15 characters
        • Password should contain atleast 1 capital letter
        • Password should contain atleast 1 digit
        • Password should contain atleast 1 special character, space or an emoji
  • Password strength: 1
    • This type of strength is displayed when 3 characteristics are met.
    • Note: If language is case sensitive then both capital and small letters check is included. Else only small letter check is included among the two
    • Example: abPQ123
    • Code sample:
      • const result = checkStrength('abPQ123');
      • console.log(result);
      • Output:
        • Strength: 1
        • Password should contain minimum of 8, and maximum of 15 characters
        • Password should contain atleast 1 special character, space or an emoji
  • Password strength: 2
    • This type of strength is displayed when all characteristics are met.
    • Note: If language is case sensitive then both capital and small letters check is included. Else only small letter check is included among the two
    • Example: abPQ123#$
    • Code sample:
      • const result = checkStrength('abPQ123#$');
      • console.log(result);
      • Output:
          Strength: 2
